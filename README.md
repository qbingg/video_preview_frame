# video_preview_frame
该项目用于快速且无损的获取视频某一帧的截图
## 用法
克隆仓库：
```
git clone https://github.com/qbingg/video_preview_frame.git
cd video_preview_frame
```
用QtCreator打开：
```
video_preview_frame.pro
```
## 项目结构
MyDisplayImgWidget.h
main.cpp
main.py
mainwindow.cpp
mainwindow.h
mainwindow.ui
video_preview_frame.pro

## 需要环境
Qt6
ffmpeg7.0
